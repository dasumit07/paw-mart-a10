PawMart is an online pet adoption and marketplace platform where users can adopt pets, list their own pets, or sell pet-related products.  
The app is built with **React** and **Tailwind CSS** featuring modern UI, smooth animations, and responsive design.

---

## Features

- **Dynamic Pet Listings:** Users can add new listings for pets or products and view othersâ€™ listings.  
- **Category-Based Filter:** Easily filter listings by category (Pets, Food, Accessories, Care Products, etc.).  
- **Dynamic Page Titles:** Each route automatically updates the browser title dynamically.  
- **Framer Motion Animations:** Smooth fade-in and scroll animations for better UX.  
- **SweetAlert2 Notifications:** Beautiful confirmation and success alerts for user actions.  
- **Responsive UI:** Fully optimized layout for mobile, tablet, and desktop screens.  
- **PDF Export Feature:** Generate downloadable PDF reports using jsPDF and AutoTable.
